# BBAFilm Website

A modern, responsive portfolio website for BBAFilm - film production and creative services.

**Live Demo:** https://bbc-antigravity.babarshah0525.workers.dev/  
**Built with:** HTML5, CSS3, Vanilla JavaScript  
**Built as part of:** [Google x Kaggle 5-Day AI Agents Intensive: Vibecoding Course](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google)

### **📸 Screenshots**

![Homepage Screenshot](./images/screenshot-home.png)

### **✨ Features**
- **Fully Responsive** - Works on mobile, tablet, and desktop
- **Fast Loading** - Pure HTML/CSS/JS, no frameworks or build step
- **Portfolio Gallery** - Showcase film projects with image grid
- **Contact Form** - Get inquiries from potential clients
- **Smooth Animations** - CSS transitions for professional feel
- **SEO Optimized** - Semantic HTML and meta tags
- **Deployed on Edge** - Hosted on Cloudflare's global network

### **🛠️ Tech Stack**
- **Frontend:** HTML5, CSS3, JavaScript ES6+
- **Deployment:** Cloudflare Pages
- **IDE:** Antigravity IDE
- **Version Control:** Git + GitHub

### **📂 Project Structure**
```
bbc-website-using-antigravity/
│
├── index.html          # Main homepage
├── style.css           # All styling and responsive rules  
├── app.js              # JavaScript for interactivity
├── images/             # All site images and assets
│   ├── logo.png
│   ├── hero-banner.jpg
│   └── portfolio/
└── README.md           # Project documentation
```

### **🚀 Deployment**

This site is deployed on **Cloudflare Pages** for free global hosting.

**Option 1: Cloudflare Pages via GitHub**
1. Fork this repo
2. Go to Cloudflare Dashboard → Workers & Pages → Create → Pages
3. Connect to Git → Select this repo
4. Build settings: Framework = `None`, Build command = blank, Output directory = `/`
5. Deploy

**Option 2: Direct Upload**
1. Download this repo as ZIP
2. Go to Cloudflare Pages → Upload assets
3. Drag ZIP file → Deploy

**Option 3: Netlify Drop**
1. Drag the project folder to https://app.netlify.com/drop
2. Done

### **💻 Local Development**
No build tools needed. Just open `index.html` in your browser.

For live reload during development:
```bash
# Using VS Code Live Server extension, or:
npx serve .
```

### **🔧 How to Customize**

| File | What to change |
| --- | --- |
| `index.html` | Update text, add/remove sections, change project titles | 
| `style.css` | Change colors, fonts, spacing. Main brand color is in `:root` | 
| `app.js` | Add contact form logic, new animations | 
| `images/` | Replace with your own photos/logos |
